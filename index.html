web design

User experience design (UX design): UX design is focused on creating a website that is easy to use and navigate. It involves considering the needs and goals of users, and designing the website in a way that helps them achieve those goals.

graphic design

Web graphic design: Web graphic design is focused on creating digital graphics for websites. This can include things like logos, icons, and images.

responsive design

Bootstrap: Bootstrap is a CSS framework that provides a number of features that can be used to create responsive designs. Foundation: Foundation is another CSS framework that provides a number of features for responsive design. Tailwind CSS: Tailwind CSS is a utility-first CSS framework that can be used to create custom responsive designs.

Coding

Coding is the process of writing the code that makes a website work. Web designers typically need to have some knowledge of coding languages such as HTML, CSS, and JavaScript. In addition to these skills, web designers also need to be familiar with the latest web design trends and technologies. They also need to be able to collaborate with other web professionals, such as front-end developers and back-end developers.

Adobe Creative Suite
